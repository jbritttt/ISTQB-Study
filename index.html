What is acceptance testing?

Testing to see if it meets the customers requirements

What is unit testing?

Testing individual components

What is alpha testing?

Testing from development environment

What is beta testing?

Beta testing is testing done by the customer in there own environment

What is regression testing?

Testing everything after making changes

What is maintenance testing?

Testing the system after deployment

What is confirmation testing?

Testing that the change has worked

What is integration testing?

It's testing multiple parts of the application to make sure they work together

What is systems testing?

It focuses on the overall behaviour and capabilitys of an entire system or product

What is System integration testing?

It's testing the interface of the system under test and other systems and external services

What is component integration testing?

(also known as unit integration testing ) focuses on testing the interfaces and interactions

What is functional testing?

Testing the functions of a component/system for correctness, are they appropriate and achieve the desired outcome

What is non-functional testing?

Testing how well the component or system behaves

What is black box testing?

Testing to check behaviour against specification, doesnt require diagrams or access to code. Based on requirement specifications

What is white box testing?

Its testing based on knowledge of the code,code design, architectures and dataflows
